# Aceid.cc Mini

Aceid.cc Mini - Бесплатный чит для игры "Warface", в нём присутствует минимальный функционал, и ничего лишнего!

### Функционал чита
 * Настройка отдачи
 * Настройка разброса
 * Отключение увода
 * Отключение затвора
 * Быстрая смена оружия

 * Аимбот
 * Триггер-бот
 * Прострел стен
 * Бесконечные патроны

 * Силуэты
 * Настройка FOV персонажа
